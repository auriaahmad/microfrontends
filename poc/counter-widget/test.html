1. Auto-Init Method (Declarative)

Widgets are automatically initialized using data attributes on elements with class counter-widget.

<div class="counter-widget" data-label="Auto Counter" data-initial="10" data-color="#2196F3" data-size="medium"></div> <script src="http://.26.251:3004/dist/counter-widget.js"></script>

2. Manual Init Method (Programmatic)

Widgets are initialized programmatically using JavaScript API for more control.

<div id="manual-counter"></div> <script src="http://.26.251:3004/dist/counter-widget.js"></script> <script> CounterWidget.init('manual-counter', { label: 'Manual Counter', initialValue: 100, color: '#E91E63', size: 'large' }); </script>
